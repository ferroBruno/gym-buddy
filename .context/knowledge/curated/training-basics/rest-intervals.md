# Rest Intervals

## Use

Use for questions about rest between sets.

## Guidance

- Short rests can work for lighter or isolation exercises.
- Longer rests help preserve performance in heavier compound exercises.
- A practical general range is 1-2 minutes for lighter work and 2-3 minutes for heavier compound work.
- If technique or performance drops sharply, resting longer can be more useful than forcing the next set early.

## Boundaries

- Do not prescribe a precise rest interval as individualized programming.
- Keep the answer flexible and tied to exercise difficulty, fatigue, and technique.
