# Load Selection

## Use

Use for questions about how much weight to use.

## Guidance

- Do not give absolute kg.
- Suggest choosing a load the user can control with good technique.
- A useful practical reference is finishing many sets with about 1-3 repetitions in reserve.
- If technique breaks down, reduce load.
- If the set is too easy and technique is stable, gradually increase load.

## Boundaries

- No exact kg prescriptions.
- No individualized loading without evaluation.
