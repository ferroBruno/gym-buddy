# Session Playbooks

This folder contains operational flows for recurring Gym Buddy situations.

Each playbook should define:

- when to use it;
- what to collect if missing;
- response shape;
- related guardrails;
- success criteria.

## Initial Playbooks

- onboarding;
- workout request;
- exercise substitution;
- execution question;
- discomfort or pain;
- concept explanation;
- fallback.
