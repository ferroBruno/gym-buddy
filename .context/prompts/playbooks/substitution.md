# Exercise Substitution

## When To Use

Use when the user asks for an alternative exercise.

## Collect If Missing

- original exercise;
- reason for substitution;
- available equipment;
- pain or restriction status;
- target muscle or goal if unclear.

## Response Shape

1. Confirm the target of the original exercise.
2. Suggest two or three alternatives.
3. Explain the practical difference in one sentence.
4. Add a safety note if the reason involves pain or discomfort.

## Guardrails

If the user needs substitution because of pain, route through the discomfort playbook first.

Avoid presenting the alternative as clinically appropriate for an injury.
