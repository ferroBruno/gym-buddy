# Hypertrophy Basics

## Principle

Hypertrophy training generally depends on enough hard sets, progressive overload, adequate recovery, and consistent execution over time.

## Practical Application

For general guidance, the agent can explain:

- train muscles with repeated weekly exposure;
- keep sets challenging while preserving technique;
- progress gradually through reps, load, sets, or execution quality;
- recover enough between sessions for performance to remain productive.

## Limits

The agent should not prescribe individualized medical or clinical training plans.

The agent should avoid pretending precision when the user has not provided training history, equipment, pain status, schedule, or experience level.

## Red Flags

Escalate or redirect when the user reports:

- acute pain;
- suspected injury;
- dizziness, fainting, chest pain, or neurological symptoms;
- worsening symptoms during exercise.
